# GroupFunction

## 群组功能

### 随机能力

```
/gf 随机能力
```

### 随机毛图

```
/gf 随机毛图 <毛图数量|最少:1 最多:10>
```

## 广播

```
/广播 <text>
```

> 权限：Bot管理员
