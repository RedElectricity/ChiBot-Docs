## Arcaaa账号管理

### 绑定账号

```
/aaa bind <ArcaeaID>
```

### 取消绑定账号

```
/aaa unbind
```

## Arcaaa 分数查询

### Arcaaa 最近分数

```
/arc info
```

需要绑定账号

会有点慢，请不要重复查询

## Arcaaa 记忆原点兑换

```
/aaa mmr <记忆原点数量>
```

>  需要先在UserCenter注册账号
