# 用户中心

## 账号操作

### 注册

```
/user reg
```

### 查看信息

```
/user
```

### 更改用户等级

```
/user setlevel <UserLevel>
```

> 权限：Bot管理员

| 用户等级 | 详细        |
| -------- | ----------- |
| 0        | Banned 用户 |
| 1        | 普通用户    |
| 2        | 超级用户    |

## 获取coin

### 每日签到

```
/user sign
```

每日零点开始可以签到

### 使用礼包码

```
/user code <礼包码>
```

> 只能领取一次

### 增加礼包码

```
/user code-add <code> <次数> <coin数> <EXP数>
```

> 权限：Bot管理员
