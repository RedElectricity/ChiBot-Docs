# Stock 股票

## Stock 股票账号

### 注册账号

```
/stock reg
```

> 需要先在UserCenter注册账号

### 查看账号

```
/stock info
```

> 需要注册账号

## Stock 查看股票

```
/stock
```

每一分钟更新价值

## Stock 股票操作

### 买入股票

```
/stock buy <股票ID> <股票数量>
```

> 需要拥有 UserCenter 和 StockMarket 账号

### 卖出股票

```
/stock sell <股票ID> <股票数量>
```

> 需要拥有 UserCenter 和 StockMarket 账号
